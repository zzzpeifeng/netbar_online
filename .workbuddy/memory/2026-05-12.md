# 2026-05-12 工作记录

## 修复图表首次加载空白问题（OnlineRateCompare.vue）

**问题现象**：首次进入页面，图表门店筛选默认选中吉姆电竞相关门店，但图表区域空白，需要手动在筛选中点一次才会显示。

**根因分析（竞态 + 异步未等待）**：
1. `fetchData()` 数据加载完成后调用 `updateChart()`（漏写 await）
2. `updateChart()` 发现 `chartInstance === null`，调用 `initChart()`
3. 旧版 `initChart()` 内部用 setTimeout 重试，属于"fire and forget"，`updateChart()` 不会等它完成
4. `initChart()` 延迟完成后，没有再触发 `updateChart()`，导致图表空白

**修复内容**：
- `initChart()` 改为返回 Promise，内部用 `requestAnimationFrame` 轮询容器可见性，可见后 resolve(true)
- `updateChart()` 中 `await initChart()` 确保初始化真正完成后再渲染数据
- `fetchData()` 中 `updateChart()` 改为 `await updateChart()`
- `onMounted` 中移除独立的 `setTimeout initChart`（已不再需要）
